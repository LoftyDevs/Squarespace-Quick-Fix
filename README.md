## CSS Quick Fix
``` CSS
/* Website Page Transition */
@keyframes SlideIn {
   0% {
   transform: translateX(+100%);
   }
   100% {
   transform: translateX(0); 
  } 
}

#page{
 animation: 1s ease-out SlideIn!important;
 border-left: 1px solid #CFFF81 
}
```

``` CSS
/* Website Pages Border */
body {
 border: 5px solid #CFFF81 
}
```
